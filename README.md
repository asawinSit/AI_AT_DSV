# Artificiell intelligens — VT2026 Stockholm Universitet

---

## Dependencies

För att köra detta måste man ha **Unreal Engine 5.7** nedladdat.

Inga utomstående programbibliotek behövs.

## Avändning

1. Öppna projektet i **Unreal Engine**, kan öppnas via .uproject filen
2. Klicka på `Play`

## Kontroller

| Input                   | Handling              |
| ----------------------- | --------------------- |
| `w`                     | Gå fram               |
| `s`                     | Backa                 |
| `a`                     | Rotera vänster        |
| `d`                     | Rotera höger          |
| `q`                     | Växla mellan AutoMove |
| `p`                     | Pausa                 |
| `esc`                   | Stänga av             |
| `Vänster klick på tank` | Skjuta                |
